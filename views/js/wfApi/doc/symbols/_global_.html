Initialize the interfaces communication for the context recovery. The source service defines where and how we retrieve contexts. The destination service defines where and how we send/save the contexts.
Defined in: api.js.
Parameters:
{Object} source
{String} source.type Optional, Default: "sync"
the type of source (sync|async|manual)
{Object} source.data Optional
For the manual source type, set direclty the context data structure
{String} source.url Optional, Default: "/wfEngine/RecoveryContext/retrieve"
For the sync and async source type, the URL of the remote service
{Object} source.params Optional
the parameters to send to the remote service
{String} source.format Optional, Default: "json"
the data format. Only json is supported in the current version
{String} source.method Optional, Default: "post"

{String} destination.method Optional, Default: "post"
